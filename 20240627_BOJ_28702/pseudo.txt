    만약 s가 숫자라면:
        s4 = s + 떨어진 칸 갯수 // 예: s==s1 이라면 s4 = s + 3
        반복 탈출
    아니라면:
        건너뛰기
반복 종료

만약 s4 % 15 == 0 이라면:
    s4 = "FizzBuzz"
아니라면 만약 s4 % 3 == 0이라면:
    s4 = "Fizz"
아니라면 만약 s4 % 3 == 0이라면:
    s4 = "Buzz"
